wf_reportR Documentation

Generate Walk-Forward Report

Description

Prints a concise summary of a wf_optimization_result: configuration, stitched OOS performance, and parameter stability.

Usage

wf_report(wf, digits = 4)

Arguments

wf

A wf_optimization_result object (from run_walk_forward()).

digits

Integer; number of digits when printing numeric values (default 4).

Value

Invisibly returns the optimization summary data frame.
