combine_scoresR Documentation

Combine multiple score panels (mean/weighted/rank-average/trimmed)

Description

Combine multiple score panels (mean/weighted/rank-average/trimmed)

Usage

combine_scores(
  scores_list,
  method = c("mean", "weighted", "rank_avg", "trimmed_mean"),
  weights = NULL,
  trim = 0.1
)

Arguments

scores_list

List of wide score panels to combine.

method

combination method.

weights

Optional numeric weights for method='weighted'.

trim

Trim fraction for method='trimmed_mean'.
